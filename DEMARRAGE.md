# Démarrage MKUBE Planning

## 1. Installer les dépendances

```powershell
npm install
```

## 2. Lancer l'application

```powershell
npm start
```

Un QR code s'affiche dans le terminal.

## 3. Tester sur votre téléphone

1. Ouvrez **Expo Go** sur votre téléphone
2. Scannez le QR code
3. L'app se charge en quelques secondes

## Structure des onglets

| Onglet | Contenu |
|--------|---------|
| 📅 Planning | Vue semaine par collaborateur |
| 🏗️ Chantiers | Liste + fiche détail |
| 👷 Équipe | Horaires + absences |
| ⚙️ Réglages | Gestion collaborateurs + export |

## Données de test incluses
